Project Overview

A production-ready ETL pipeline demonstrating data engineering best practices for defense asset telemetry and incident management. This project showcases the complete data lifecycle: from heterogeneous ingestion to star-schema warehousing and automated ML integration.

Specifically designed for defense and aerospace use-cases, the pipeline consolidates data from:

  • Relational Ops DB: Asset lifecycle, event logs, and incident reports.
  • CSV Log Extracts: Batch data from legacy external systems.
  • JSON Telemetry: Real-time sensor feeds and platform monitoring.
